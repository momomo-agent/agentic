# 器官手册

agentic 家族的每个模块对应 AI 的一个"器官"。

| 器官 | 模块 | 职责 |
|------|------|------|
| 🧠 大脑 | [core](/packages/core) | LLM 调用基础 |
| 👁️ 眼睛 | [sense](/packages/sense) | MediaPipe 感知 |
| 💪 意志 | [act](/packages/act) | 意图→决策→执行 |
| 🎙️ 声音 | [voice](/packages/voice) | TTS + STT |
| 💾 记忆 | [memory](/packages/memory) | 短期+长期记忆 |
| 🦴 骨骼 | [store](/packages/store) | SQLite 持久化 |
| 📝 表达 | [render](/packages/render) | Markdown 渲染 |
| 🔗 嵌入 | [embed](/packages/embed) | 向量化 |
| 📁 文件 | [filesystem](/packages/filesystem) | 虚拟文件系统 |
| ⚡ 命令 | [shell](/packages/shell) | 命令执行 |
| 🗺️ 空间 | [spatial](/packages/spatial) | 空间推理 |
| 🦾 身体 | [claw](/packages/claw) | 完整 agent 运行时 |
