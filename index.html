Philosophy
Why this exists

No abstraction tax

Each module is a single file you can read in 10 minutes. No hidden complexity, no magic. When something breaks, you can find and fix it yourself.

Composition over framework

Use one module or all eight. They work together by convention, not coupling. Replace any piece with your own implementation — the others won't notice.

Browser-first

Every module works with a script tag. No Node.js required, no build step, no transpiler. Drop it in and it works.

AI is the platform

Not a chatbot library. A toolkit for building AI-native interfaces — where the AI senses, thinks, decides, and acts as a first-class participant.
